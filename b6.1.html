What is CSS?

Cascading Style Sheets(CSS) is used to format the layout of the web page

With CSS, you can control the color, font, the size of test, the spacing between elements, how are positioned and layout, what background images or background colors are to be used, different displays for different devices and screen size, and much more!

Tip: The word cascading means that a style appiled to a parent element will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs and other text elements within the body will who also get the same color (unless you specify something else)!
